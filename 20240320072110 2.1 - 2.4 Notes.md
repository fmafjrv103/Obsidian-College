# Pre-Calc Notes 2.1 - 2.4
## Organizing input/output values
- Table of Values (T-Chart)
Ex:

|                |     |
|:--------------:| --- |
|       3        | -5  |
| $-\frac{7}{3}$ | 16  |
|      0.8       | 0   |
- Ordered Pair
	(3, -5)
	($-\frac{7}{3}$, 16)
	(0.8, 0)
- Function Notation
	$f(3)=-3$
	$f(\frac{7}{3})=16$
	$f(0.8)=0$
## Vertical Line Test
A graph is a function if any vertical line drawn through the graph intersects the graph no more than once.
